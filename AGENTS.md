# AGENTS.md - Baǵdar

## Setup

### Backend
```bash
cd backend
python -m venv .venv
.venv\Scripts\activate  # Windows
pip install -r requirements.txt
# Create .env with GEMINI_API_KEY=your_key
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

### Frontend
```bash
cd frontend
npm install
npm run dev  # http://localhost:5173
```

## Commands

- `npm run build` - Frontend build to `dist/`
- `npm run lint` - ESLint
- No typecheck command in frontend (tsc -b is in build)

## Architecture

- **Frontend**: React 19 + Vite + Tailwind 4 + PWA plugin
- **Backend**: FastAPI + Gemini AI (Google GenerativeAI SDK)
- API base URL: `http://localhost:8000` (dev)

## Key Files

- Frontend entry: `frontend/src/main.tsx`
- Backend entry: `backend/app/main.py`
- API routes: `backend/app/api/v1/endpoints/`
- Generate endpoint: `POST /api/v1/generate`

## Gotchas

- Backend requires `.env` with `GEMINI_API_KEY` - no fallback
- Frontend uses path alias `@` -> `frontend/src`
- PWA manifest generated by vite-plugin-pwa in `vite.config.ts`
- Frontend port 5173, Backend port 8000
